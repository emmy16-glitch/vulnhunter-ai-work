# Information Architecture

The main navigation follows the operator's governed data flow:

`Overview → Collection → Analysis → Independent Review → Governance → Intelligence → Assurance`

Detail screens are contextual and do not occupy permanent navigation slots. The
role-filtered sidebar only shows areas a user may enter, while direct route access
must still be rejected by the backend when the role is not authorized.

The campaign control room is the central operational object. It connects approved
intent, registered applications, linked scans, review coverage, adjudication,
release blockers, and the final manifest without duplicating the underlying source
records.
